라이선스 매니저 구성

사용 중인 플랫폼에 따라 다음과 같이 여러 가지 방법으로 라이선스 매니저를 구성할 수 있습니다.

Windows

라이선스 매니저 서비스를 구성해 두면 시스템 부팅 시 라이선스 매니저가 자동으로 시작되고, Windows 서비스 제어판을 통해 라이선스 매니저의 관리를 중앙 집중화할 수 있습니다. 라이선스 매니저 디폴트 구성 설정을 검토하려면 디폴트 설정 보기를 클릭하십시오. 라이선스 매니저 서비스를 구성하지 않도록 선택하는 경우 나중에 구성할 수 있습니다.

    참고:   기존 라이선스 매니저가 있어 새 라이선스 매니저를 설치하기 전까지 계속 사용하고자 하는 경우에는 라이선스 매니저를 서비스로 구성하지 않음(파일은 설치됨) 옵션을 선택합니다. 인스톨러는 라이선스 매니저 파일 설치를 계속합니다. 설치를 마치면 기존 라이선스 매니저를 중지하고 새 라이선스 매니저를 시작할 수 있습니다.

Linux

시스템의 시작 항목 폴더에 라이선스 매니저에 대한 항목을 추가하려면 이 옵션을 선택된 채로 두고 다음을 클릭합니다. 라이선스 매니저 프로세스를 소유할 사용자 이름을 지정합니다. 보안상 이 이름은 루트일 수 없습니다.
